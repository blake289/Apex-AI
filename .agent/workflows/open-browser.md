# Opening Websites in Browser

When opening any website or local file in the browser, always use Safari.

## Steps

1. Use the `open` command with the `-a Safari` flag to open URLs/files in Safari:
   ```bash
   open -a Safari "URL_OR_FILE_PATH"
   ```

## Examples

- Local HTML file: `open -a Safari "file:///path/to/file.html"`
- Web URL: `open -a Safari "http://localhost:3000"`
